My Projects

  1. Composite Laminate Analysis Code - Wrote a program in MATLAB/Python to conduct basic laminate analysis on advanced composite materials. Parameters that can be edited include the material properties, applied loads, thermal/hydrogroscopic effects, and orientation of the plies. Currently working on creating a GUI with a dropdown menu for materials in US and SI units, along with a function for progressive failure of the laminate.

  2. Wind Turbine Project - I was the project manager, and directed design, analysis, and manufacturing of the turbine components. I created the timeline for the project and established the procedure for Vaccum-assisted Resin Transfer Method (VARTM) turbine blade fabrication. Additionally, I conducted the analysis of the turbine blades using hand calculations and finite element analysis (FEA) software for natural frequencies and dynamic wind loading scenarios. By identifying manufacturing constraints in the procedure, I designed the tooling to contain two blades instead of one, decreasing the fabrication time by half.
